What is the Document Object Model (DOM)?

The DOM helps make websites interactive and more interesting to use. It turns the page into a tree of objects, where everything like text, images, and buttons are parts of this tree. By using JavaScript, you can change these partsâ€”like adding new text, changing colors, or hiding buttons.

How do you select an element from the DOM?

There are multiple ways to select an element from the DOM using JavaScript,anyone can use getElementById('idName'), getElementsByClassName('className'), and querySelector('selectorType') or querySelectorAll('selectorType') to select elements using CSS-style selectors.These methods help access webpage elements.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique in the DOM where a parent element handles events for its child elements. Instead of adding event listeners to each child, you add one to the parent. This is useful because it saves memory, simplifies code, and works even for dynamically added child elements.

How do you manipulate an element's attributes and styles using the DOM?

Anyone can manipulate an element's attributes using setAttribute() to change things like id or class.To change styles,you modify the style property,such as element.style.backgroundColor = "blue";.This allows you to update how the element appears or interacts with users dynamically.
